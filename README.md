# JNI 函数签名对应表  

| 签名 | 对应类型 |
| --- | --- |
| L&lt;java class&gt;; | Java类，如：Ljava/lang/String; |
| Z | Boolean |
| B | Byte |
| C | Character |
| S | Short |
| I | Integer |
| J | Long |
| F | Float |
| D | Double |
| V | Void |
